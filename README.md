# 💳 ATM Transaction System

## 📌 Project Description
This project is a Java-based ATM simulation system. It allows users to perform basic banking operations like withdrawing money, depositing money, checking account balance, and PIN validation.

---

## 🚀 Features
- 🔐 Secure PIN validation (3 attempts)
- 💰 Deposit money
- 💸 Withdraw money
- 📊 Check balance
- 🚪 Exit option

---

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Console Application

---

## ▶️ How to Run
1. Install Java (JDK)
2. Open terminal / command prompt
3. Compile the program:
   ```bash
   javac ATM.java
